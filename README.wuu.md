## 我儂自家嗰物事

#### 我儂嗰shell腳本

* `alias.sh`: 命令嗰別名
* `custom.sh`: 環境變量
* `functions.sh`: 嘸畀用嗰物事
* `cow.sh`: 用cowsay隨機講一句閒話
* `pony.sh`: 用ponysay隨機講一句閒話
* `cow_or_pony.sh`: 用cowsay或ponysay隨機講一句閒話 = =
